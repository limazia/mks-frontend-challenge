### Rodando a aplicação

```bash
# Clone o repositório
$ git clone https://github.com/limazia/mks-frontend-challenge

# Acesse a pasta
$ cd mks-frontend-challenge

# Instale as dependências do projeto
$ npm install ou yarn

# Rode a aplicação
$ npm run dev ou yarn dev
```
